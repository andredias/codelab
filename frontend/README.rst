Configuration
=============

This project version is based on `vite <https://vitejs.dev/>`_.

Commands:

* ``npm install``
* ``npm run dev``
* ``npm run build``
* ``npm run preview``


Environment Variables
=====================

``.env.development``:

.. code:: console

   VITE_API_URL=http://localhost:5000


``.env.production``:

.. code:: console

   VITE_API_URL=/api
