1. Coletar os termos:

    $ pybabel extract -F babel.cfg -o translations/message.pot .

2. Atualizar as traduções (talvez seja necessário usar o Python 2.7 para isso)

    $ pybabel update -i translations/message.pot -d translations/

3. Compilar

    $ pybabel compile -d translations/
